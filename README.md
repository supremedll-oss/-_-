# 可编辑分镜脚本工作台

打开 `index.html` 即可使用，不需要安装依赖。

## 当前版本

- 首页：默认进入瀑布流项目库，可新建、打开、删除项目。
- 品牌风格：首页已加入几力 Studio Logo，整体采用青柠绿、奶油白、淡黄和黑色粗描边的工作室风格。
- Logo 资源：站点使用 `assets/jili-studio-logo.png`，发布到 GitHub Pages 时需要一起上传 `assets` 文件夹。
- 项目卡：保留黑胶专辑陈列形式，封面和唱片优先使用项目中的第一张分镜画面；没有分镜画面时显示黑色磨砂质感。
- 项目编辑器：左侧为项目名和统计，右侧为分镜时间线。
- 时间线：镜头块可拖拽换序，末尾虚线镜头块可新建镜头，选中镜头后按 Delete 可删除。
- 时间线缩放：时间码表支持滚轮缩放，镜头块会随时间精度变化伸缩，短镜头也会继续缩短。
- 节点画布：每个镜头包含分镜画面大节点，以及景别、画面描述、镜头语言、表达含义四个默认小节点；画布支持滚轮缩放和鼠标中键拖动。
- 节点图片：分镜画面、参考画面、参考效果通过“添加图片”按钮上传，支持 GIF，并按图片比例显示。
- 自动排布：点击节点工具栏里的“自动排布”可整理画布，小节点会避开重叠。
- 灵感收集器：保存网页链接、标签、类型、难度圆点和图片/GIF。
- AI 视频提示词收集：提示词全项目通用，可拖到节点画布生成带分类的小节点，也可拖到输入框并自动追加 `；`。
- 界面整理：节点画布采用紧凑网格工作区，提示词支持去重和删除，全站统一黑色描边与几力 Studio 主题色。
- 回撤：编辑器内支持 Ctrl+Z 回撤当前项目的上一步操作。

## 保存方式

编辑内容会自动保存在当前浏览器的本地存储中。换浏览器、换设备或清理浏览器数据后，本地记录可能消失，建议定期导出 JSON 工程备份。

旧版单项目数据会自动迁移为首页里的第一个项目。
