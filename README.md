# 可编辑分镜脚本工作台

打开 `index.html` 即可使用，不需要安装依赖。

## 新版结构

- 首页：默认进入瀑布流项目库，可新建、打开、删除项目；项目以黑胶专辑形式展示，封面和唱片会优先使用项目里的分镜画面，没有分镜画面时显示黑色磨砂质感。
- 项目编辑器：左侧为项目名和统计，右侧为分镜时间线。
- 时间线：镜头块可拖拽换序；末尾虚线镜头块可新建镜头；选中镜头后按 Delete 可删除。
- 节点画布：每个镜头都有深色节点式分镜画布，默认包含分镜画面大节点，以及景别、画面描述、镜头语言、表达含义四个小节点。
- 节点连线：点击一个节点连接点，再点击另一个节点连接点即可连线；点击连线可取消连接。
- 自动排布：点击节点工具栏里的“自动排布”可整理画布。
- 灵感收集器：保存网页链接、标签、类型和难度圆点。
- AI视频提示词收集：按核心主体、特定场景、叙事动作、时长、运镜类型、景别构图、布光方案、核心色调分类；提示词可拖到节点画布生成小节点，也可拖到输入框并自动追加 `；`。

## 保存方式

编辑内容会自动保存在当前浏览器的本地存储中。换浏览器或清理浏览器数据后，本地记录可能消失，建议定期导出 JSON 工程备份。

旧版单项目数据会自动迁移为首页里的第一个项目。
